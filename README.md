# Pastebin-
A ready to use Pastebin written in Go Lang

Specialities:-
•Get Custom Links
•Gets file secured
•use get method to get data from the pastebin

How to get started:-
•Fork the Git
•Install all the modules with go get method
•go run main.go

How to access?
•/main to create paste
•/view to view files
•/edit to edit files
